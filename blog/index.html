我的数码生活（一）

序

对于懒惰的我，坚持写博客真的似乎是不可能的事。总是想写些东西，记录自己百无聊赖的生活，这生活如此乏味，我自己不记录恐怕永远都会消失在时间里，等到在我笨笨的脑袋慢慢地淡忘往事，它们就真的凭空消失了，一点痕迹都没有留下，无影无踪，无处找寻，想到这里，背脊一阵发凉，于是赶紧勤劳起来，记录下还没消失的生活。

既然要写，总要有主题，能唤起我书写欲望的，你们也看到了，就如题目写的：我的数码生活。

起始

我何时开始对数码感兴趣，要追溯到高中时代。高二前，当我还没遇到改改之前，根本不知数码为何物。那是2008年，iPhone也才在前一年发布，诺基亚还是曾经的巨头，嘲笑着苹果的玩物手机。安卓还是一个带着全键盘的怪胎，黑莓也像诺基亚一样风头正劲。世界上还没有移动互联网这个词汇，用手机最多也是最潮的似乎就是上QQ，一切看起来都像是移动互联网的蛮荒时代。当然，我就像当时的诺基亚一样浑然不知，就像不懂NBA的女孩子不会知道挡拆、扣篮这些词汇的意义一样，我也不会知道苹果黑莓这些词汇除了水果之外的别的意义。

你不了解

背景

这是我高四（是的，没错！就是高四！！）高考完之后开的博客里的文章，当时开博客是因为当时迷恋的女孩子开了博客，想想距今已有五年。再写到这里，保存一下我年轻的文字。

你不了解

今天很糟，我很不高兴，我不知道怎么了，心中像堵了一大块石头，把心事都憋在里面，倒不出愁，也添不进喜，也许是太闲了，闲的我瞎想，想东想西，思南思北，没有头绪。

放假一个多月了，看着别人旅游的旅游，打工的打工，而我呢，闲着，睡觉，看电视，上网。。。。都特别没意思。只是午夜的聊天是唯一的盼望吧，还有她还有些希望。但当一切变了，也许迫不得已，也许厌倦了，也许就到了变化的时候，大家都朝着自己的目标大步迈进，而我还在广平小城，有一些悲凉的，有一些无奈的，有一些不甘的，过着，抑或，苟延残喘。

超哥说过，我颠倒整个世界，只为摆正你的倒影。我想，但受困。我要颠倒世界，摆正的或许是你背影吧，远离的背影。当一切变成无言，城与城之间的九十公里，便成为了沉默的汪洋，泛着悲情的蓝色，飞翔着孤单的海鸥，咸咸的海风里，是我眼泪的味道。
我知道了博客的作用了，在你不在了，我可以想着你，抒发内心的喜与愁，高兴与悲伤，微笑与眼泪。他很费时间，而我现在最重要的就是消磨时间，哈哈。

我要干些什么呢，来度过一天中没你的二十三个小时。。。。。。

2015-03-26

杭州的春天

杭州的春天来了。

经过了寒冷的难熬的寒冬，春天的到来更显得弥足珍贵，冬天阴冷暗淡的太阳不知被谁杀死了，而新上任的太阳是一位青春妙龄的少女，温柔曼妙，婀娜多姿，总是让人忍不住得要待在她的怀抱里，醉死在温柔乡。来自北方的我小时候读课文总是读到“春天来了，到处生机勃勃”，可现实里却是寒冷的冬天慢吞吞得赖着不走，夏天就气势汹汹的来了。春天总是匆匆，仿佛夹在中间受气的小姑娘。人们还没来得及感受到处的生机勃勃，就被粗鲁的夏天轰出了人间，让人猝不及防。但杭州的春天不一样，她像一幅画卷，在慢慢地展开，不急不缓，让人领略到每一处的风采，享受到每一处的滋味，沉浸入春天的温暖幸福的氛围里。

第一次感受到杭州的春天是在一个早晨，住在三楼的我还没睡醒，就隐约感到拉住的窗帘都挡不住的灿烂的阳光，睁开眼看，窗帘已被照耀成一股温柔的暖黄色，从拉紧的窗帘的缝隙还偷偷钻进一束阳光，照到我的被子上，成一块光斑，我不禁用手去触碰，冬日沁凉的被子的感觉已消失的无影无踪，入手一片温暖，顿时暖进了心底里，暖得我睡意全消，于是伸懒腰，起床。

起来就迫不及待的拉开窗帘，嚇！一片温柔的金黄色，阳光照耀在每一处楼房，由远及近，从高到低。我的屋子里立刻便被灌满了温暖的感觉，就这么突然间，冬日的阴沉消失了，春天就这么duang的一下来了！路上的行人似乎也比往日多了，大家都嘴角上扬，看起来都和我一样沉浸在一片春意中。而马上我的注意力就被楼下一片白色吸引，楼下两座单云楼之间种着两颗树，我学识浅薄，不知是什么树，但现在满满当当都开出了白的的花，从我三楼的视角俯视好不壮观，但花刚刚开，还透着一股娇嫩。微风吹来，满树的花朵随风微微摇晃。我立刻打开了窗户，微风拂面，是春姑娘温柔的抚摸吧，一股花香也随风飘进了房间，是春姑娘清新的体香吧。

突然想起一句诗“忽如一夜春风来，千树万树梨花开”，这是白的梨花形容瑞雪，但现在的情景却是“就是一夜春风来，一树两数梨花开”，现在，我确信，是的！春天来了！是的！夏天还远！我迫不及待的出门了，去到这杭州美丽的春天里了。

记录博客发布流程

距离上一次写博客又过去了差不多两个月，有一个笑话是程序员花了两天的时间各种搞，终于搭起了自己的个人博客，然后再也没更新过。。。我现在就变成了这个笑话。因为更新不频繁，所以每次更新都要百度一下hexo的指令，所以索性在这里记录一下，下一次直接看这篇东西就好了。

准备

首先hexo是要求node环境的，我的电脑是已经安装了node，所以直接跳过这一步。GitHub方面我也是准备好的，所以万事俱备，只欠“hexo”。

Hexo安装与配置

npm install hexo -g #-g表示全局安装, npm默认为当前项目安装
Hexo使用命令：
hexo init <folder> #执行init命令初始化hexo到你指定的目录
hexo generate #自动根据当前目录下文件,生成静态网页
hexo server #运行本地服务
浏览器输入http://localhost:4000就可以看到效果。

添加新文章

hexo new "postName" #新建博文,其中postName是博文题目
博文会自动生成在博客目录下source/_posts\postName.md
#文件自动生成格式:
title: "It Starts with iGaze: Visual Attention Driven Networkingwith Smart Glasses" #博文题目
date: 2014-11-21 11:25:38 #生成时间
tags: Paper #标签, 多个标签使用格式[Paper1, Paper2, Paper3,...]
---
如果不想博文在首页全部显示, 并能出现阅读全文按钮效果, 需要在你想在首页显示的部分下添加<!--more-->

主题更改

Hexo提供了官网的主题, 初始化hexo时也会自动生成一个主题, Hexo还支持个性定制主题, 可以根据自己的喜好对主题进行修改, 更多主题可以在官网中找到

个性化博客的设置

在博客的根目录下对喜爱的主题进行主题进行克隆
git clone git@github.com:yunlzheng/hexo-themes.git themes/writing

#在./_config.yml，修改主题为writing
theme: writing

#查看本地效果
hexo g #hexo generate简写
hexo s #hexo server简写

部署到Github

以上内容都是在本地进行查看, 现在将博客部署到github上
打开./_config.yml
在配置文件中设置号部署设置后,
hexo deploy #进行部署

总结

经过一番梳理，大概的流程是这样的：

  • 打开Ulysses,尽情挥洒文采（呵呵呵）
  • 保存到相应文件夹
  • hexo g
  • hexo s 本地预览

终于有耐心再写点东西

终于有耐心再写点东西，上一篇看日期还是2014年，现在已经是2015年的一月下旬了，匆匆之间，又过了一年。

时光流逝

最近在亚马逊上买书，看到出版日期是2012年，2013年的书，下意识里还觉得好新的书，可细想一下已经是三四年前了，再想一步我也不再是三四年前的毛头小子了。可我的心态似乎一点都没变，还是个孩子的心态。这是想起小时候总是会想，我长大了是不是就会变成爸爸那样的大人，变得不苟言笑，变得威严肃穆。小时候的印象里的大人好像都是那个样子的，现在自己真的变成了所谓的“大人”，觉得自己和小时候也没什么差别。

在我小时候，爸爸就和我现在的年纪差不多，是不是也像我现在一样懵懂不知世事，忧愁着别人觉得不值一提的小事，憧憬着似乎还布满迷雾的未来。而当二十年后，我变成了爸爸现在的年纪，会不会也变得不苟言笑，威严肃穆，心里会不会还是一股孩子气呢？

跨年

今年第一次跨年，往年的一月一日都如往常，觉得跨年然后通宵是个很愚蠢的行为，今年就轮到了自己愚蠢。

其实是很开心的，吃烤肉，聊天，开玩笑。去ktv，认识新朋友，一开始拘谨，后来热络，听着别人唱好听的歌，由着自己把好听的歌唱得很难听，然后，过一整夜。

我很喜欢的妹子跟我说，她要分手了。（我喜欢妹子，妹子不喜欢我，剧情总是这样的。）我嘴上说最好不要分。心里管理妄想的部门却想着还不赶紧分，但心里还有一个明辨是非的部门告诉我，她只是说说而已。

我看着她，她的双颊微红，显得令人心醉。ktv很吵，环境嘈杂，我贴着她的耳朵对她说话，她贴着我的耳朵对我说话，声音沙哑，喉咙发疼，可一切都那么舒服。

第二天，2015年1月1号，昨天是一场梦吧。

美剧

复仇那个美剧还是没看下去，这几天倒是续上了行尸走肉，上个周末的时间从第三季中间，一直看到了第五季，还是很好看。

发现丧尸片如果是电影可以只和丧尸斗，像我是传奇，但电视剧如果每集都砍丧尸剧情就太单薄了，还是人和人斗能牵起剧情高潮，所以丧尸片也只是在丧尸快要占领地球，灭亡人类的时候，演绎的人和人的组织内部矛盾。

阅读

最近看完了《侠客行》，突然对金庸的武侠有点饱和的感觉，于是立即调转方向，找了好久，找到一本叫《时光之轮》的外国科幻小说，评价都很高，篇幅也很长，现在在慢慢看，还没有看出门道。

还买了一本书《编写可维护的JavaScript》，43块钱的书，我以为会很厚，结果到了一看才200页，很薄的书，顿时有种上当的感觉。希望是本对我有用的书。

我一直很喜爱的播客《糖蒜广播》最近出书了，为了纪念糖蒜十年，这必须买，分分钟亚马逊下单，明天应该就可以到。

过年

还有不到一个月就过年了，真是很想家，希望时光快些，早点可以回家。

2014-12-15日，记

今天是周一。

美剧

周末两天在租的房子里待了两天没出门，因为身体不舒服，要静养。看了一个美剧，叫《复仇》，感觉没有想象中有意思，我还是喜欢像《国土安全》这一类型的美剧，情节一以贯之，节奏紧张，应该是上周末，看了第四季《国土安全》的前九集，真的是一口气看完，实在是过瘾，但再看别的就有点食之无味的感觉。
总之现在还在找一个新的美剧，让我沉浸其中，不能自拔。

关于美剧，我最近比较喜欢的几部有疑犯追踪，国土安全，权利的游戏，绝命毒师，纸牌屋。现在的新剧感觉都一般。

阅读

周六的时候还把《碧血剑》看完了，我的宏大的“金庸作品重读计划”已经完成了《鹿鼎记》和《碧血剑》，下一部准备看《侠客行》。很明显，《鹿鼎记》比《碧血剑》好看得多，感觉后者还是很稚嫩的，还在老派武侠的感觉里没走出来，前者则是金庸后期非常成熟的作品。
